快速幂

求解\(a^b\)% m问题
1.常规写法
基本数学公式：a b % m = (a % m) (b % m)
引申：\(a^b\) % m = \((a \% m)^n\)

2.递归写法
如果b为偶数，a^b = a^(b/2) a^(b/2)
如果b为奇数，a^b = a a^(b - 1), b - 1为偶数
如果b == 0，递归结束

3.迭代写法
把b写成二进制，假设b为十进制13，二进制为1101，13 = \(2^3+2^2+2^0\)，\(a^b = a^8 a^4a^1\)。
令ans = 1；
每次迭代判断b的二进制数末尾是否为1，为1的话乘以a并对m取模；
令a平方，b右移一位
如果b>0，返回第二步迭代
